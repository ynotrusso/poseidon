<hr>

### Installation

```
npm i -g poseidon-js
```

### Package.json

```
npm init -y
```

### Generate a Server

```
ps gs
```

### Generate a Database

```
ps gd
```

### Generate a Model

```
ps gm
```

### Generate a Controller

```
ps gc
```
